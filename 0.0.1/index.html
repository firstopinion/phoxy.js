Getting Started

phoxy.js is a client library to preview and upload images to phoxy server.

Package Managers

npm: npm install phoxy.js

phoxy

Client library to preview and upload images to phoxy server.

configure

Configure the phoxy client.

Function
configure(url, credentials)

Parameters

Name Type Description
url string The root url of the phoxy service.
credentials object Phoxy service credentials.
credentials
Name Type Description
id string Phoxy authentication id.
secret string Phoxy authentication secret. 
phoxy.configure('PHOXY_URL', {
  id: 'PHOXY_ID',
  secret: 'PHOXY_SECRET'
})

create

Create a phoxy image instance. A phoxy image instance takes an image file, optionally resizes it, and exposes an interface for both previewing and uploading the image.

Function
create(file, options, callback)

Parameters

Name Type Description
file file File (https://developer.mozilla.org/en-US/docs/Web/API/File)
options object Image options
callback function Callback once image has been either created or has thrown an error.
options
Name Type Description
maxWidth number Maximum width of generated image (will be resized proportionally).
maxHeight number Maximum height of generated imaged be resized proportionally). 
phoxy.create(file, {
  maxWidth: 500
}, function (err, img) {
  // upload? render?
});

upload

Upload an image to a phoxy compatible server.

Function
upload(image, callback)

Parameters

Name Type Description
image object Phoxy image instance to upload.
callback function Function to execute once image upload has completed. 
phoxy.uplad(image, function (jqXHR, textStatus) {
  // response handling
});

Test

All dependencies must be installed (npm install)

Tests also require the following env variables to be set:

  • PHOXY_URL
  • PHOXY_ID
  • PHOXY_SECRET

Run: npm test

Contributing

In lieu of a formal styleguide, take care to maintain the existing coding style. Add unit tests for any new or changed functionality. Lint and test your code using Grunt.

Also, please don't edit files in the "dist" subdirectory as they are generated via Grunt. You'll find source code in the "src" subdirectory!
